ChatGPT TFG Prompt Generator for Engineers


Este bookmarklet proporciona un conjunto de prompts generadores diseñados específicamente para ingenieros que están desarrollando su Trabajo Fin de Grado (TFG).
Cada prompt está pensado para asistir en diversas etapas del proyecto, desde la investigación inicial hasta la revisión y defensa final.

Para instalar arrastra el botón a la barra de marcadores del navegador.
En dispositivos móviles pulsa el botón.
Si no está visible, para activar barra de favoritos pulsa CONTROL + Mayúsculas + B.
